# Stock Portfolio Tracker 📈

A simple Python-based Stock Portfolio Tracker that calculates the total investment value of stocks entered by the user.

---

## 📌 Project Description

This project allows users to:

- Enter stock names and quantities
- Calculate investment values
- Display portfolio summary
- Save the report into a text file

The project uses hardcoded stock prices and demonstrates basic Python programming concepts.

---

## 🚀 Features

- User-friendly console interface
- Predefined stock price dictionary
- Calculates total investment value
- Displays portfolio summary
- Saves report into `.txt` file
- Beginner-friendly Python project

---

## 🛠 Technologies Used

- Python 3
